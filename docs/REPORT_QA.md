# 指导书问题与报告回答落点

本文档不是最终答案全文，而是给报告生成器和后续写报告的人提供一个“问题清单”。

每一条都标明了来源：

- `指导书原问`：指导书中明确提出的问题、要求解释的内容、要求分析的内容
- `报告补充分析`：虽然不一定是指导书原句，但属于报告里最好补上的分析点，避免只写现象、不做解释
