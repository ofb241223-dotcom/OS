# 操作系统课程设计仓库说明

本仓库用于保存《2026年操作系统课程设计指导书》对应实验的源代码、可执行文件与报告支撑材料。

## 目录结构

- `windows/src/`：Windows 实验源代码
- `windows/output/`：Windows 实验编译后的可执行文件
- `linux/src/`：Linux 实验源代码
- `linux/output/`：Linux 实验编译后的可执行文件
- `docs/`：报告撰写辅助材料
- `RUN.md`：按指导书顺序整理的编译与运行命令
- `2026年操作系统课程设计指导书.pdf`：任务书原文

## 实验范围

- A1-A5：验证性实验
- B1-B9：综合性实验

## 报告相关材料

`docs/` 中已经按“报告怎么写”而不是“代码怎么放”来整理，适合直接给网页版 ChatGPT 读取并生成报告草稿。

建议优先阅读：

1. `docs/REPORT_WRITING_GUIDE.md`
2. `docs/REPORT_QA.md`
3. `docs/SCREENSHOT_PLAN.md`
4. `docs/REPORT_PROMPT.md`

## 说明

- 报告应以第一人称撰写。
- 封面信息建议手动填写。
- 图片截图建议由本人实际运行后补充。
